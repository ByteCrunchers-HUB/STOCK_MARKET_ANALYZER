📈 Stock Market Analyzer using NumPy

A beginner-friendly data analysis project built using NumPy, focused on understanding real-world stock market behavior through vectorized computations and time-series analysis.

Instead of just practicing theory, this project applies NumPy to actual financial data to simulate how real data analysts and quantitative traders explore stock trends.

🚀 Features
📊 Stock Filtering

Select specific stocks (e.g., AAPL, AMZN, MSFT) from a mixed dataset using boolean indexing.

📉 Daily Return Analysis

Calculates day-to-day percentage change in stock prices to track profit/loss movement.

📈 Average Return

Computes overall average return to understand long-term growth trends.

⚡ Volatility Analysis

Measures how much a stock price fluctuates to estimate risk and stability.

📉 Moving Average Trend

Smooths noisy data using rolling averages to identify general upward or downward trends.

🔁 SMA Crossover Signals

Compares short-term and long-term Simple Moving Averages (SMA) to generate basic BUY/SELL signals.

📊 RSI-Based Insights

Uses Relative Strength Index (RSI) to detect overbought or oversold market conditions.
